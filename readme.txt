# SomeBlog
SomeBlog theme is specially made for bloggers so they can easily showcase their blog posts. We made sure that the texts are easy to read with the right font size and spacing. It is a minimal and elegant theme. Perfect for people that want to keep it simple.

# Installation #
1. After downloading the zip file, go to Appearance > Themes and click on the "Install Themes" tab.
2. Click on the "Upload" link.
3. Upload the zip file that you downloaded and click "Install Now".
4. Click "Activate" to use the theme you just installed.


# Theme Customizer #
With the Theme Customizer, you can set your site title and tagline, set a background image, set a custom color, assign your menus, and choose a static home page. You can preview your changes by clicking the Customize link below your active theme on your Appearance > Themes page.


# Widgets #
SomeBlog supports widgetized areas in the sidebar and footer.


# Responsive #
SomeBlog is fully responsive. Your content will look good whether it's viewed on a desktop, a tablet or a smartphone.
